### SpringBoot解决的痛点，解决了**配置地狱问题**，Spring各种的配置问题，整合Spring技术栈。
- 自动Start依赖，简化构建配置
- 自动配置Spring以及第三方配置
- 能够提供生产级别的监控，健康检查及其外配置 （方便后期运维）
